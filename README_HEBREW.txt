CASPER AI - גרסה נקייה ונוחה

מה השתנה:
- אין עיצוב האקר.
- אין מסך טעינה ארוך.
- עיצוב נקי, מהיר ונוח.
- צ'אט AI בעברית.
- מצב חיפוש באינטרנט.
- מצב הסבר פשוט.
- מצב רעיונות.
- מצב עזרה בלימודים.
- היסטוריית שיחות נשמרת בדפדפן.
- מצב בהיר ומצב כהה.
- כפתורים מהירים לשאלות נפוצות.
- עובד גם בטלפון.

הפעלה:
1. התקן Node.js.
2. חלץ את קובץ ה-ZIP.
3. שנה את שם .env.example ל-.env
4. הכנס את מפתח ה-API שלך:
   OPENAI_API_KEY=המפתח_שלך
5. הפעל את Start_Casper_AI.bat
6. האתר ייפתח בכתובת:
   http://localhost:3000

חשוב:
- אל תשתף את מפתח ה-API שלך.
- השימוש ב-OpenAI API עשוי לעלות כסף.
